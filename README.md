# Keploy Tutorial — Echo + PostgreSQL

A single-page documentation site built with Next.js and MDX, explaining how to use Keploy to record and replay API tests for a Go URL Shortener app.

## Live Demo
[keploy-tutorial-flax.vercel.app](https://keploy-tutorial-flax.vercel.app)

## Tech Stack
- Next.js 16 + MDX
- Tailwind CSS
- Custom Callout components

## Run Locally
```bash
npm install
npm run dev
```
